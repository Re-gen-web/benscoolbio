## Installation

Add this line to your Jekyll site's Gemfile:

```ruby
gem "jekyll-athena"
```

And add this line to your Jekyll site `config.yml`:

```yaml
theme: jekyll-athena
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jekyll-athena
